Bootstrap Utilities

Borders

Bootstrap border utilities provide ways to style the border and border-radius. Learn More.

Colors

Change the borders colors using the built bootstrap theme colors (.primary, .secondary, etc.). 


                <span class="border border-primary"></span>
                <span class="border border-secondary"></span>

Border-radius

Round the borders of an element by adding different classes. 


                <span class="border rounded"></span>
                <span class="border rounded-circle"></span>
