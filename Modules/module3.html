Bootstrap Components

Alerts

Provide contextual feedback messages for typical user actions with the handful of available and flexible alert messages. Learn More.

Example

Alerts are available for any length of text, with an optional dismiss button. Proper styling, requires one of the eight contextual classes (e.g., .alert-success, alert-primary, alert-secondary).

This is a primary alert—check it out! 
                    
                        <div class="alert alert-primary" role="alert">
                                This is a primary alert—check it out!
                        </div>

Links

For links use the .alert-link utility class to provide a matching color links within any alert.

This is a success alert with an example link. Give it a click if you like. 
                    
                        <div class="alert alert-success" role="alert">
                            This is a success alert with 
                            <a href="#" class="alert-link">an example link</a>. 
                            Give it a click if you like.
                        </div>

Dismissing

Using the alert JavaScript plugin, it’s possible to dismiss any alert inline. Here’s how:

  • Be sure you’ve loaded the alert plugin, or the compiled Bootstrap JavaScript.
  • Add a dismiss button and the .alert-dismissible class, which adds extra padding to the right of the alert and positions the .close button.
  • On the dismiss button, add the data-dismiss="alert" attribute, which triggers the JavaScript functionality. Be sure to use the <button> element with it for proper behavior across all devices.
This is a secondary alert with the optional dismissal—check it out! 
                    
                        <div class="alert alert-warning alert-dismissible fade show" role="alert">
                            This is a secondary alert with the optional dismissal—check it out!
                        <button type="button" class="close" data-dismiss="alert" aria-label="Close">
                            <span aria-hidden="true">&times;</span>
                        </button>
                        </div>

Cards

A flexible and extensive container of bootstrap. It provides options for headers and footers a wide variety of content, contextual background colors, and powerful display options. Learn More.

Content Types

Cards support a wide variety of content, including images, text, list groups, links, and more. Below are examples of what’s supported.

  • Card headers can be styled using .card-header class within the < h*> tag
  • Card titles can be styled using .card-title class within the < h*> tag
  • .card-img-top puts an image on top of the card
  • Text can be added to a card using the .card-text class
  • The building block of a card is the .card-body. Use it whenever you need a padded section within a card.
  • Card footers can be styled using .card-footer class within the < h*> tag

Navbar

A guide for bootstrap's navigation header and navbar. Learn More.

Example

Bootstrap's navbar consists of different sub-components:

  • .navbar-brand for your company, product, or project name.
  • .navbar-nav for a full-height and lightweight navigation (including support for dropdowns).
  • .navbar-toggler for use with our collapse plugin and other navigation toggling behaviors.
  • .form-inline for any form controls and actions.
  • .navbar-text for adding vertically centered strings of text.
  • .collapse.navbar-collapse for grouping and hiding navbar contents by a parent breakpoint.
